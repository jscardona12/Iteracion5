Para el uso de nuestra aplicación, es necesario contar con las siguientes herramientas:
	Postman (Complemento de Google Chrome)
	Eclipse Mars 3.0
	Wildfly 8.0
	JDK 1.8.0.77
	SQL Developer

1) Descargue el proyecto 
2) Importe el proyecto a Eclipse
3) Ejecute Postman
4) Ejecute el proyecto en el servidor WildFly 8.0 dentro de Eclipse
5) Genere una sentencia dentro de Postman (Puede encontrar sentencia para probar los requerimientos 
					   funcionales en el archivo adjunto postman.txt)
6) verifique que las operaciones fueron realizadas correctamente dentro de la base de datos.
